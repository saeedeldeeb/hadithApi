Hadith API Documentation

The following endpoints are supported by the Hadith API. They all support the HTTP GET method and return JSON.

GET

All these endpoints give you a JSON object describing an edition languages.

Endpoint
Description
Example
https://ahadith-api.herokuapp.com/api/language
Lists all languages in this edition are available
https://ahadith-api.herokuapp.com/api/language
GET

All these endpoints give you a JSON object all books available in our database.

Endpoint
Description
Example
1. https://ahadith-api.herokuapp.com/api/books/{{language}}
Lists all available books. You can filter the results using the parameters below.

Parameters

language - A 2 digit language code. Example: 'en', 'ar'.

  1. https://ahadith-api.herokuapp.com/api/books/ar
  2. https://ahadith-api.herokuapp.com/api/books/en
GET

All these endpoints give you a JSON object listing all chapters with differ responses.

Endpoint
Description
Example
1. https://ahadith-api.herokuapp.com/api/chapter/all/{{language}}
Lists all available chapters. You can filter the results using the parameters below.

Parameters

language - A 2 digit language code. Example: 'en', 'ar'.

  1. https://ahadith-api.herokuapp.com/api/chapter/all/ar - Lists all chapters in Arabic.
  2. https://ahadith-api.herokuapp.com/api/chapters/all/en - Lists all chapters in English.
2. https://ahadith-api.herokuapp.com/api/chapter/{{bookid}}/{{language}}
Lists all chapters in which Book are available.
https://ahadith-api.herokuapp.com/api/chapter/10/ar - Chapters in book 10 in Arabic. https://ahadith-api.herokuapp.com/api/chapter/1/en - Chapters in book 1 in English.
GET

All these endpoints give you a JSON object describing Chapter Introduction.

Endpoint
Description
Example
1. https://ahadith-api.herokuapp.com/api/chapter/intro/{{bookid}}/{{chapterid}}
Gets chapter Introduction. You can filter the results using the parameters below.

Parameters

  • bookid - Specify a book.
  • chapterid - Specify the chapter.
  • https://ahadith-api.herokuapp.com/api/chapter/intro/1/6 - Gets chapter 6 intro in book 1.
GET

All these endpoints give you a JSON object listing Ahadith.

Endpoint
Description
Example
1. https://ahadith-api.herokuapp.com/api/ahadith/all/{{language}}
Lists all available ahadith. You can filter the results using the parameters below.

Parameters

    language
  • en - For English
  • ar-tashkeel - For Arabic with tashkeel.
  • ar-noTashkeel - For Arabic without tashkeel.
  1. https://ahadith-api.herokuapp.com/api/ahadith/all/en - Lists all Ahadith in English.
  2. https://ahadith-api.herokuapp.com/api/ahadith/all/ar-tashkeel - Lists all Ahadith in Arabic with tashkeel.
  3. https://ahadith-api.herokuapp.com/api/ahadith/all/ar-notashkeel - Lists all Ahadith in Arabic without tashkeel.
2. https://ahadith-api.herokuapp.com/api/ahadith/{{bookid}}/{{chapterid}}/{{language}}
Lists all Ahadith in which chapter are available
https://ahadith-api.herokuapp.com/api/ahadith/1/1/ar-notashkeel - List Ahadith of Chappter 1 in book 1.
GET

All these endpoints give you a JSON object listing all Ahadith that have your word.

Endpoint
Description
Example
1. https://ahadith-api.herokuapp.com/api/ahadith/search/{{keyword}}/{{language}}
Lists all available Ahadith with this keyword.
  1. https://ahadith-api.herokuapp.com/api/ahadith/search/Messenger/en - Lists all Ahadith with "Messenger" keyword.
  2. https://ahadith-api.herokuapp.com/api/ahadith/search/الله/ar-notashkeel - Lists all Ahadith with "الله" keyword.
